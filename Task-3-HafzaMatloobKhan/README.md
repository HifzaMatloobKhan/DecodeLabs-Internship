Task 3 – SQL Data Analysis
Objective

The objective of this task is to perform structured data analysis using SQL queries on the imported dataset.
The task focuses on extracting meaningful insights such as sales trends, product performance, payment behavior, and order statistics.
It helps in understanding how to analyze real-world datasets using SQL for business decision-making.

Dataset

The dataset used in this task was imported into SQL Server from a CSV file and converted into a structured relational table.
It contains order-related information such as product details, payment methods, order status, and total price values.
The dataset was validated and used for performing analytical SQL queries to generate insights.

Tools Used

SQL Server Management Studio (SSMS) was used to execute all SQL queries and manage the database environment.
SQL (Structured Query Language) was used for data retrieval, filtering, aggregation, and analysis operations.
The imported dataset was stored in a SQL database table for structured querying and analysis.

Output

Successfully executed SQL queries to analyze total sales, average order value, product performance, and payment distribution.
Identified key insights such as top-selling products, order status trends, and high-value transactions using aggregation queries.
The final output demonstrates the ability to perform data analysis using SQL and extract meaningful business insights from raw data.
