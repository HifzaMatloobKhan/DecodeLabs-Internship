Task 2 – Exploratory Data Analysis (EDA)
Objective

The objective of this task is to perform Exploratory Data Analysis on the cleaned dataset to identify patterns, trends, and data distributions.
The analysis focuses on understanding customer behavior, product trends, payment preferences, and order statistics using Python.
This task also includes calculating statistical measures and detecting outliers to gain meaningful business insights from the dataset.

Dataset

The cleaned dataset generated from Task 1 was used for the analysis process to ensure accurate and reliable results.
The dataset was examined to study numerical values, categorical distributions, transaction behavior, and product-related trends.
Using the cleaned dataset helped improve the quality of insights and reduced inconsistencies during the analysis stage.

Tools Used

Python was used to perform data analysis, statistical calculations, and trend identification efficiently.
Pandas library was utilized for data manipulation, descriptive statistics, filtering, and outlier detection operations.
Jupyter Notebook was used to execute the analysis workflow, display outputs, and document observations professionally.

Output

Successfully performed Exploratory Data Analysis by calculating mean, median, count values, and analyzing dataset distributions.
Identified customer and product trends, payment method preferences, order status distribution, and unusual outlier records.
The analysis generated meaningful insights from the dataset and produced a structured workflow ready for reporting and further analytics.
