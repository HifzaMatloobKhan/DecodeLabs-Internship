Task 2 – Exploratory Data Analysis (EDA)
Objective

The objective of this task is to perform exploratory data analysis on the dataset to understand its structure, patterns, and key trends.
The analysis focuses on identifying relationships between variables and detecting important insights from the data.
This step helps in forming a clear understanding of the dataset before applying advanced analytics or visualization techniques.

Dataset

The dataset used in this task is the raw sales dataset provided for the internship project.
It includes multiple fields such as product details, customer transactions, quantities, prices, and dates.
Initial inspection was performed to understand data distribution, missing values, and overall structure.

Tools Used

Python was used as the main programming language for performing exploratory data analysis tasks.
Pandas library was used for data loading, filtering, grouping, and statistical summary calculations.
Basic data exploration functions were applied to understand trends, distributions, and relationships within the dataset.

Output

The dataset was analyzed to extract key insights such as sales trends, product performance, and customer behavior patterns.
Summary statistics and grouping operations were used to identify important data distributions and relationships.
The analysis provided a strong foundation for further visualization and advanced data processing tasks.
