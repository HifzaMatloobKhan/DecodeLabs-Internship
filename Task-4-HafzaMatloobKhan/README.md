Task 4 – Data Visualization and Analysis
Objective

The objective of this task is to analyze the cleaned dataset using data visualization techniques in Python.
The goal is to identify patterns, trends, and relationships between different variables such as sales, quantity, and product performance.
Multiple charts are created to transform raw data into meaningful insights for better decision making.

Dataset

The dataset used in this task is the cleaned version of the original sales dataset prepared in previous steps.
It contains structured records including product details, quantities, prices, dates, and payment methods.
This cleaned dataset ensures accurate visualization and reliable insights without data inconsistencies.

Tools Used

Python is used as the core programming language for performing all visualization tasks.
Pandas library is used for data manipulation and preparation of aggregated views for charts.
Matplotlib and Seaborn libraries are used for creating various types of graphs such as bar charts, line plots, scatter plots, heatmaps, and histograms.

Output

Multiple visualizations were generated including bar charts, line graphs, scatter plots, stacked charts, heatmaps, and distribution plots.
The analysis revealed key insights such as top-performing products, monthly revenue trends, and correlation between quantity and total price.
All graphs were saved as image files and used to understand customer behavior and sales performance effectively.
