Task 1 – Data Cleaning and Preprocessing
Objective

The objective of this task is to clean and preprocess the raw dataset using Python to improve overall data quality and reliability.
The task focuses on identifying inconsistencies, handling missing values, removing duplicate entries, and validating incorrect formats.
This preprocessing step ensures that the dataset becomes accurate, structured, and ready for further analysis and reporting.

Dataset

The dataset provided for the internship task contains raw records with possible inconsistencies and incomplete information.
The data was carefully inspected to identify null values, duplicate rows, incorrect date formats, and structural issues.
A detailed preprocessing workflow was followed to transform the dataset into a clean and analysis-ready format.

Tools Used

Python was used as the primary programming language for performing all data cleaning operations efficiently.
Pandas library was utilized for data manipulation, preprocessing, validation, and handling missing or duplicate values.
Jupyter Notebook was used as the development environment to execute the workflow, analyze outputs, and document the process.

Output

The dataset was successfully cleaned by removing duplicate entries and handling missing or invalid data values.
Date formats and dataset structure were verified to ensure consistency and improve data accuracy for future analysis.
The final cleaned dataset was exported and saved properly, making it ready for visualization, reporting, and advanced analytics tasks.
